Boids Module

This module implements a flocking simulation using the Boids algorithm, leveraging KAK decomposition.

Source Code

The main source file for this module is modules/boids/boids_main.scm.

API

run-boids

Signature: (run-boids args)

Run the Boids simulation. args is a list of command-line arguments.

main

Program entry point. Calls run-boids with command-line arguments.
